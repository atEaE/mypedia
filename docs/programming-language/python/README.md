## 【関数に対する引数の指定方法】

関数に対する引数の指定方法は以外と言語による差がある部分。  
加えて、引数の数が多い場合や可変長引数を許可している関数の場合は、可読性にも直結するので引数の指定方法は覚えておいて損はなし。

```py
# 仮引数を指定する方法
def add(a, b, c):
    d = a + b + c
    return d

# 仮引数=実引数という書き方ができる
print(add(a="Hello", b="World", c="!!"))

# or 

# 仮引数を指定した場合、引き渡し順は何でも良い。
print(add(b="World", c="!!", a="Hello"))
```
